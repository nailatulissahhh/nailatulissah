OSI LAYER

OSI Layer adalah sebuah konsep yang memungkinkan pertukaran informasi terjadi antara berbagai jenis sistem komunikasi komputer, dengan menggunakan protokol standar, yaitu TCP/IP.

Manfaat Osi Layer

01

Memudahkan untuk memahami cara kerja jaringan komputer secara menyeluruh.

02

Memungkinkan vendoripaker untuk mendesain dan mengembangkan hardware atau software yang sesuai dengan layer tertentu.

03

Breakdown persoalan komunikasi data untuk mempermudah troubleshooting.

Fungsi Osi Layer

Fungsi dari konsep OSI layer adalah memudahkan proses pencarian titik awal permasalahan, sehingga meminimalkan waktu yang di perlukan untuk melacak masalah jaringan. Dengan begitu, masalah jaringan bisa diatasi dengan lebih mudah.

Berikut adalah tujuh model OSI layer yang di mana setiap lapisannya memiliki fungsi dan tugas masing-masing. 

Application Layer (Lapisan ke 7)

pusat terjadinya suatu interaksi antara user dengan aplikasi yang bekerja menggunakan fungsionalitas sebuah jaringan.

Presentation Layer (Lapisan ke 6)

untuk mengidentifikasi sintaks yang di pakai suatu host jaringan untuk berkomunikasi. 

Session Layer (Lapisan ke 5)

untuk mengendalikan dialog maupun melakukan pengelolaan terhadap koneksi suatu komputer.

Transport Layer (Lapisan ke 4)

Memecahkan data yang akan dimasukkan ke dalam beberapa paket data.

Network Layer (Lapisan ke 3)

untuk mendefinisikan alamat IP sehingga setiap komputer dapat saling terkoneksi dalam satu jaringan. 

Data Link Layer (Lapisan ke 2)

untuk memeriksa bila terjadi kesalahan dalam menyalurkan transmisi terhadap bit data.

Physical Layer (Lapisan pertama)

sebagai transmisi terhadap bit data.
